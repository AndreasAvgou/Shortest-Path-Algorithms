# Shortest Path Algorithms

This package contains:

- graph_algorithms.py
- graph_generator.py
- main.py
- measurements.csv (created after running)
- plot.png (created after running)

## Requirements
Python 3.10+
pip install matplotlib

## Run

```
python main.py
```

The program will:
1. Generate graphs
2. Run Dijkstra & Bellman-Ford with different representations
3. Save measurements to CSV
4. Generate plot.png
